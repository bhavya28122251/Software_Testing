# Student Management App
This scaffold contains a backend (Node + Express + SQLite using better-sqlite3) and a frontend (Vite + React).

## Quick start
1. Backend:
   - `cd backend && npm install`
   - `npm run seed` (optional demo data)
   - `npm start` (runs on http://localhost:4000)
2. Frontend:
   - `cd frontend && npm install`
   - `npm run dev` (open Vite dev server URL)
